Finding the right blood type during an emergency can be challenging. **BloodConnect** bridges the gap between willing donors and patients. This frontend UI provides a streamlined, accessible interface for users to register as donors, search for specific blood types in their city, and view critical information regarding rare blood groups. 

This project is built with a mobile-first approach, ensuring that users can access life-saving connections on any device.
